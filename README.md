# 🌟 Креативный сайт с анимацией скролла

![Image](https://github.com/user-attachments/assets/d5a94f5b-b6d7-4b50-becd-5c46a6e87685)

Этот проект — одностраничный веб-сайт с использованием библиотек **GSAP ScrollTrigger** и **GSAP ScrollSmoother**. Главная фишка — плавная прокрутка и анимации, которые оживают при скролле, создавая эффект погружения.

## 🚀 Технологии

В проекте используются следующие инструменты:

- **GSAP (GreenSock Animation Platform)** — профессиональная библиотека для создания высокопроизводительных анимаций.
  - **ScrollTrigger** — позволяет привязывать анимацию к прокрутке страницы.
  - **ScrollSmoother** — создаёт эффект "плавного" скролла с инерцией и сглаживанием.

## 💻 Локальный запуск

Для запуска проекта на локальном компьютере рекомендуется использовать **Live Server**:

1. Установите расширение **Live Server** в Visual Studio Code
2. Откройте папку с проектом в VS Code
3. Нажмите правой кнопкой мыши на файл `index.html`
4. Выберите **"Open with Live Server"**
5. Проект автоматически откроется в браузере с поддержкой автоматического обновления при изменениях кода

Альтернативный способ — просто открыть файл `index.html` в любом современном браузере.
